What Is This

A field guide format for navigating contested territory

Every domain has a full landscape: rigorous science, promising-but-early research, popular belief, and outright mythology. Most resources pick a lane — debunker or believer. Spectrum Docs try to show the whole gradient without flattening it.

Each doc places items on a tiered evidence spectrum with color-coded badges, short descriptions, and mechanism boxes — the star feature — which explain what's actually claimed or measured, in plain language.

Docs come in two scales: macro (broad domain, many items, lightly covered) and micro (one narrow topic, deep parameters). Items can appear in multiple docs, contextualized differently each time.

format · single HTML file · no build step · Google Fonts only
tiers · each doc sets its own tier count and labels — visual language stays consistent
tone · informative, not preachy · honest about evidence · respectful of the unexplained
Spectrum Docs are a reference series, not medical or financial advice.
Evidence quality assessments reflect the state of published research — not a claim that things work or don't work.
